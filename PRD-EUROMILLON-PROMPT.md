# 🎯 PROMPT PARA CHATGPT - PRD EUROMILLÓN MÁGICO

## 🎯 INSTRUCCIONES PARA CHATGPT

Actúa como un Product Manager experto en apps de lotería y gamificación. Necesito que crees un PRD (Product Requirements Document) completo y detallado para una app de Euromillón con el mismo enfoque mágico y medieval que Amuleto Bonoloto, pero adaptado a las características específicas del Euromillón (5 números + 2 estrellas).

## 🎯 CONTEXTO DEL PROYECTO

### 📱 App Referencia: Amuleto Bonoloto
- **Tema**: Medieval esotérico con orbes mágicos
- **Motores**: 4 algoritmos diferentes (histórico, Buffon, dados, altar personal)
- **Tecnología**: Web versión con HTML/CSS/JavaScript (puro, sin frameworks)
- **Monetización**: AdMob + donaciones
- **Almacenamiento**: LocalStorage (offline-first)
- **Acelerómetro**: Interacción física en juegos

### 🎯 Objetivo: Euromillón Mágico
- **Misma temática**: Medieval esotérico con orbes y rituales
- **Adaptación**: 5 números (1-50) + 2 estrellas (1-12)
- **Misma tecnología**: Web versión pura sin Android Studio
- **Misma monetización**: AdMob + donaciones voluntarias

## 🎯 REQUISITOS ESPECÍFICOS PARA EUROMILLÓN

### 🎲 Formato Euromillón:
- **Números principales**: 5 números entre 1-50
- **Estrellas**: 2 números entre 1-12
- **Premios**: Desde 2 estrellas hasta 5+2 aciertos
- **Sorteos**: Martes y viernes (Europa)

### 🔮 Motores Algorítmicos Adaptados:
1. **Motor Histórico**: Basado en estadísticas de Euromillón
2. **Motor Buffon**: Experimento adaptado para 5+2
3. **Motor Dados**: 5 dados + 2 estrellas especiales
4. **Motor Amuleto**: Datos personales + zodiaco europeo

### 🌟 Temática Europea Medieval:
- **Caballeros**: Espadas, escudos, armaduras
- **Castillos**: Torres, mazmorras, salones del trono
- **Artefactos**: Corona real, cetro mágico, gema europea
- **Símbolos**: Cruzadas, heráldica, blasones nobles
